# CanSat

## Overview

***

Project for the [CanSat](https://www.esa.int/Education/CanSat) satellite.

## Requirements

***

1. CMake
2. Ninja
3. gcc-arm-none-eabi

## Install

***

Download the source code from GitHub and run the following command:

``` console
$ git clone https://github.com/Filipeak/trash-can.git
```
