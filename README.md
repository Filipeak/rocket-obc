# Project Rocket

## Overview

***

Project for the rocket system.

## Required Software

***

1. CMake
2. Ninja
3. ARM Toolchain
4. A compiler (GCC, Clang, MinGW, etc.)

## Install

***

1. Download the source code from GitHub or run the following command:

``` console
$ git clone https://github.com/Filipeak/project-rocket.git
```

2. Run setup script:

``` console
$ ./setup.sh
```
